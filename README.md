# Deployment of AI Applications for Cloud and Mobile Platforms

A 10-day training program presented by **HERE AND NOW AI** for **Vels Institute of Science, Technology & Advanced Studies (VISTAS)**.

## Overview
This repository contains the syllabus and resources for the 10-day intensive training program on deploying AI applications across cloud and mobile platforms.

## Schedule
- **Dates:** 2nd Feb 2026 to 13th Feb 2026
- **Time:** 9:30 AM to 12:30 PM (Daily)
- **Duration:** 3 hours per day

## Topics Covered
1. Deploying AI applications on cloud platforms
2. Understanding containers and basic DevOps concepts
3. Managing costs and performance
4. Connecting AI backends with mobile applications
5. Understanding real-world mobile AI use cases
6. API-driven cloud-to-mobile AI architecture

## Features
- **Mobile Responsive:** Optimized for all screen sizes.
- **AI Chatbot (Caramel):** A specialized assistant powered by Gemma-3-27b to answer program-related queries.
- **Organization Branding:** Follows HERE AND NOW AI design guidelines.

## Deployment on Cloudflare Pages
To deploy this project:
1. Connect your GitHub repository to Cloudflare Pages.
2. In the Cloudflare Dashboard, go to **Settings > Functions > Variables**.
3. Add the secret: `GEMINI_API_KEY` with your API key.
4. Redeploy.
